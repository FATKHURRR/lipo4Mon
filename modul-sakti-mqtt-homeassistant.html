Panduan Langkah Integrasi
  1. Tambahkan integrasi MQTT di Home Assistant Anda. Konfigurasikan dengan data berikut:
    Broker URL: broker.emqx.io | Port: 1883 | Username: emqx | Password: emqx.
    *PENTING: Matikan opsi "Enable discovery" pada saat instalasi integrasi.
  2. Di dalam direktori konfigurasi Home Assistant Anda, buat folder baru dengan struktur: ModulSakti/configs/
  3. Unduh berkas-berkas konfigurasi YAML di bawah ini, lalu unggah (upload) semuanya ke dalam folder ModulSakti/configs/ tersebut.
  4. Buka berkas utama configuration.yaml milik Home Assistant Anda, tambahkan kode berikut di bagian bawah, lalu simpan dan lakukan Reload All MQTT Entities / Restart HA: mqtt: sensor: !include_dir_merge_list ModulSakti/configs/
